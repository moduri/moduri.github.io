
```javascript

arr.sort(function compare(a, b) {
    return a - b;});
console.log(arr)

arr.sort(function compare(a, b) {
    return b - a;});
console.log(arr)
```

    sort를 사용한 내림차순, 올림차순
    [
      1, 2,  3,  4,
      4, 6, 31, 77
    ]
    [
      77, 31, 6, 4,
       4,  3, 2, 1
    ]
